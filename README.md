💼 Budget.OPS – Team Finance Dashboard

A modern, interactive team finance dashboard built using React (CDN), HTML, CSS, and JavaScript.
It helps teams track expenses, monitor budgets, and visualize spending in real-time.

🚀 Features
📊 Expense Tracking
Add, delete, and manage expenses
Track by category, team member, and date
💰 Budget Management
Predefined category budgets:
Salaries
Tools
Marketing
Travel
Office
Other
Real-time calculation of:
Total spent
Remaining budget
Over-budget alerts
📈 Data Visualization
Donut chart for spending breakdown
Category-wise progress bars
Monthly navigation
🎨 UI/UX Highlights
Custom animated cursor
Ripple click effects
Particle trails
Gradient UI with glassmorphism
Smooth animations & transitions
📦 Transactions Panel
Scrollable transaction list
Category icons & color coding
Delete functionality
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Library: React (via CDN)
Styling: Custom CSS (no frameworks)
Fonts: Google Fonts (Syne, DM Sans)


This project is open-source and free to use.
